# Estilo de citação (CSL)

Coloque aqui um arquivo `.csl` de **estilo limpo** (por exemplo `ieee.csl`) e
aponte para ele em `_quarto.yml`:

```yaml
csl: csl/ieee.csl
```

Baixe estilos em <https://www.zotero.org/styles> (ex.: busque por "IEEE").

> Decisão do projeto: estilo **limpo (IEEE-like), sem ABNT**.

Enquanto nenhum CSL estiver configurado, o Quarto/Pandoc usa o estilo padrão de
autor-data — suficiente para os builds de andaime.
